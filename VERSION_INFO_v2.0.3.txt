HOW TO USE THE NEW IMPORT FEATURE:
===================================

1. **Prepare Your Data**:
   - Export your hostel register to Excel format
   - Ensure columns match the supported format
   - Include student names, mobile numbers, fees, etc.

2. **Import Process**:
   - Open the application
   - Go to Students page
   - Click "Import Data" button
   - Upload your Excel/CSV file
   - Preview the parsed data
   - Click "Import" to add all records

3. **Data Format Example**:
   ```
   Sr.No: 17
   Name & Mobile: Mohd Wahid Mohd Sadique, at. By pass road rohinkhed Tq. Motala, 8482914403, 9372272981
   Wing & Room: C-06
   Class: B.Sc. Comp. I
   Date of Joining: 08.07.2025 to 09.05.2026
   Receipt No: 6774
   Receipt Date: 08.07.2025
   Reg.Fee: 500
   Room Rent: 3000
   Water & Electricity Charges: 3000
   Other Activity: 0
   Total Fees Collection: 6500
   Outstanding Fee: 4000
   Security Deposit: 1500
   ```

4. **What Gets Created**:
   - Student record with personal details
   - Payment record with fee breakdown
   - Outstanding balance calculations
   - Receipt generation capability
